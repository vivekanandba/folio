# Contributing to Folio

Follow `docs/DEV_WORKFLOW.md`. Spec → failing tests → implementation. Chrome freeze through Phase 2: no PRs whose primary change is glow, particles, or cinematic hub art.

## PR checklist

- [ ] Spec link (`specs/…`)
- [ ] Test file link (`tests/…`)
- [ ] Not chrome-only
- [ ] `npm test` green
- [ ] Content changes pass `npm run validate`
